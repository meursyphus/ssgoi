# 淡入淡出动画

淡入淡出（Fade）动画通过调整元素的透明度来创建平滑的出现或消失效果。

<TransitionDemo type="fade" />

## 使用方法

<Tabs items={[{ label: "React", value: "react" }, { label: "Svelte", value: "svelte" }, { label: "Vue", value: "vue" }, { label: "Angular", value: "angular" }]}>
  <TabPanel value="react">
    ```tsx
    import { transition } from '@ssgoi/react';
    import { fade } from '@ssgoi/react/transitions';

    function Component() {
      return (
        <div ref={transition({
          key: 'my-element',
          ...fade()

        </div>

    }

    <div use:transition={{

    </div>
    ```
  </TabPanel>

    <template>
      <div v-transition="{
        key: 'my-element',

  <TabPanel value="angular">
    ```typescript
    import { Component } from '@angular/core';
    import { TransitionDirective } from '@ssgoi/angular';
    import { fade } from '@ssgoi/angular/transitions';

    @Component({
      selector: 'app-component',
      imports: [TransitionDirective],
      template: `
        <div [transition]="fadeTransition">
          Fade me!
        </div>
      `
    })
    export class AppComponent {
      fadeTransition = {
        key: 'my-element',
        ...fade()
      };
    }
    ```

## 选项

```typescript
interface FadeOptions {
  from?: number;        // 起始透明度（默认：0）
  to?: number;          // 结束透明度（默认：1）
  spring?: {
    stiffness?: number; // 弹簧刚度（默认：300）
    damping?: number;   // 阻尼系数（默认：30）
  };
}
```

### 使用示例

```tsx
// 部分淡入淡出
fade({ from: 0.2, to: 0.8 })

// 缓慢淡入淡出
fade({ spring: { stiffness: 100, damping: 20 } })

// 反向淡入淡出
fade({ from: 1, to: 0 })
```
