# 트랜지션 플레이그라운드

SSGOI의 모든 트랜지션 효과를 실시간으로 테스트하고, 스프링 물리 설정을 조정하며, 바로 사용 가능한 코드를 생성할 수 있습니다.

<TransitionPlayground />

## 사용 방법

1. **트랜지션 선택**: 드롭다운에서 원하는 트랜지션 타입을 선택합니다.
2. **스프링 조정**: Stiffness와 Damping 슬라이더로 애니메이션 속도와 부드러움을 조절합니다.
3. **프리셋 사용**: Smooth, Normal, Fast 버튼으로 미리 정의된 스프링 설정을 빠르게 적용합니다.
4. **코드 생성**: 하단에 자동으로 생성되는 코드를 복사하여 프로젝트에 사용합니다.

## 스프링 물리 이해하기

- **Stiffness (강성)**: 값이 클수록 빠르고 강한 애니메이션
  - 낮은 값 (50-150): 느리고 부드러운 움직임
  - 중간 값 (200-350): 일반적인 속도
  - 높은 값 (400-500): 빠르고 스냅감 있는 움직임

- **Damping (감쇠)**: 값이 클수록 진동이 적고 안정적인 애니메이션
  - 낮은 값 (5-20): 탄성이 있고 바운스가 많음
  - 중간 값 (25-35): 균형잡힌 움직임
  - 높은 값 (40-50): 오버슈트 없이 부드럽게 정착

## 활용 팁

- 사용자 인터페이스 요소에는 빠른 반응성을 위해 높은 stiffness를 사용하세요.
- 배경이나 장식 요소에는 낮은 stiffness로 부드러운 효과를 주세요.
- 중요한 알림이나 경고에는 bounce 효과와 함께 사용하면 주목도를 높일 수 있습니다.
