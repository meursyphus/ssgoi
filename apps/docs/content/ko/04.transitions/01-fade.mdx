# 페이드 애니메이션

페이드(Fade) 애니메이션은 요소의 투명도를 조절하여 부드럽게 나타나거나 사라지는 효과를 만듭니다. 가장 기본적이면서도 널리 사용되는 애니메이션입니다.

## 기본 사용법

<TransitionDemo type="fade" />

```tsx
import { transition } from '@ssgoi/react';
import { fade } from '@ssgoi/react/transitions';

function Component() {
  return (
    <div ref={transition({
      key: 'my-element',
      ...fade()
    })}>
      Fade me!
    </div>
  );
}
```

## 옵션

```typescript
interface FadeOptions {
  from?: number; // 시작 투명도 (기본값: 0)
  to?: number; // 종료 투명도 (기본값: 1)
  spring?: {
    stiffness?: number; // 스프링 강도 (기본값: 300)
    damping?: number; // 감쇠 계수 (기본값: 30)
  };
}
```

## 사용 예시

### 부분 페이드

반투명에서 시작하여 약간 투명한 상태로 끝나는 페이드:

<TransitionDemo type="fade" options={{ from: 0.2, to: 0.8 }} />

```tsx
fade({ from: 0.2, to: 0.8 })
```

### 빠른 페이드

스프링 설정을 조정하여 더 빠른 애니메이션:

```tsx
fade({ spring: { stiffness: 500, damping: 40 } })
```
