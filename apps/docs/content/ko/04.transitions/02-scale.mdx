# 스케일 애니메이션

스케일(Scale) 애니메이션은 요소의 크기를 확대하거나 축소하는 효과를 만듭니다. 시선을 집중시키거나 중요한 요소를 강조할 때 유용합니다.

## 기본 사용법

<TransitionDemo type="scale" />

```tsx
import { transition } from '@ssgoi/react';
import { scale } from '@ssgoi/react/transitions';

function Component() {
  return (
    <div ref={transition({
      key: 'my-element',
      ...scale()
    })}>
      Scale me!
    </div>
  );
}
```

## 옵션

```typescript
interface ScaleOptions {
  from?: number; // 시작 크기 (기본값: 0)
  to?: number; // 종료 크기 (기본값: 1)
  axis?: 'x' | 'y' | 'both'; // 스케일 축 (기본값: 'both')
  spring?: {
    stiffness?: number; // 스프링 강도 (기본값: 300)
    damping?: number; // 감쇠 계수 (기본값: 30)
  };
}
```

## 사용 예시

### X축 스케일

가로 방향으로만 크기 변경:

<TransitionDemo type="scale" options={{ axis: 'x' }} />

```tsx
scale({ axis: 'x' })
```

### Y축 스케일

세로 방향으로만 크기 변경:

<TransitionDemo type="scale" options={{ axis: 'y' }} />

```tsx
scale({ axis: 'y' })
```

### 커스텀 크기

작은 크기에서 원래 크기로:

```tsx
scale({ from: 0.5, to: 1 })
```
